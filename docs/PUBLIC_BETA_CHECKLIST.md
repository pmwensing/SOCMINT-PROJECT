# Public Beta Checklist

## Required before beta access

- CI is green.
- Production release summary is ready.
- Release pipeline summary is reviewed.
- Security audit summary is reviewed.
- Gate enforcement summary is reviewed.
- Case/team access controls are configured.
- Billing provider configuration is validated in test mode.
- Production Docker/Tor smoke is rehearsed where applicable.
- Encrypted backup/restore drill is complete.
- Responsible-use, privacy, terms, and onboarding docs are published.

## Required before sensitive case use

- Scope and authorization are documented.
- Export preflight is enabled.
- Evidence retention plan is documented.
- Operator permissions are limited to need-to-know.
- Incident/misuse reporting path is defined.
