# v13.3 — Command Center Readiness Panel

## Purpose

Make dossier readiness actionable on the command-center page.

## Added

The operator panel now displays:

- Dossier readiness state.
- Dossier readiness label.
- Blocking items.
- Warnings.

## Value

The user can see why the dossier is blocked, draft-ready, or exported without opening the API payload.
