# v13.13 — Review Queue Action Buttons

## Purpose

Make the normalization review queue faster to operate from the UI.

## Changed

- Each review queue item now shows quick action buttons:
  - Confirm
  - Reject
  - Suppress
  - Reset

- The page now includes state filters for:
  - Unreviewed
  - Confirmed
  - Rejected
  - Suppressed

## Value

Analysts can now review normalized connector output from the visible UI instead of manually posting to the API.
