# v13.2 — Command Center Operator UI

## Purpose

This release makes the v13 operator workflow visible on the command-center page.

## Added

The command center now renders:

- Operator next best action.
- Dossier readiness state.
- Next-action reason.
- Subject-to-package workflow summary.

## Value

The landing page now shows one obvious action instead of only a broad dashboard of modules.

This is the visible UI step for the clean real-world SOCMINT workflow:

```text
subject -> seed -> collection/import -> findings -> review -> dossier -> package
```
