# v22.2 Release Package Preview and Redaction Check

Shows the exact material authorized for release from the latest export package and recipient/channel authorization.

The preview classifies included sections and attachments, exposes redaction or sensitivity blockers, and requires operator acknowledgement before recording the preview.

The review includes citation-ready narratives and findings, attachment paths and media types, classification, sensitivity matches, and redaction requirements.

Acknowledgement records the review even when blockers remain, preserving the audit trail while release readiness stays false.

The preview is created without transmitting or mutating the dossier package. Existing audit-log storage is reused, and no migration is introduced.
