Release Workflow

  1. Open /product/build-control and review readiness.
  2. Run make product-smoke.
  3. Run make product-route-smoke.
  4. Run make release-hardening-smoke.
  5. Check dossier quality with /api/v1/dossier/<subject_id>/quality-gate.
  6. Check traceability with /api/v1/dossier/<subject_id>/traceability.
  7. Commit, push, merge, and tag only when the hardening report is pass.
